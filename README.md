# Task Management API

A RESTful API for managing tasks built with Fastify, Sequelize, and TypeScript.

## Features

- User authentication with JWT
- CRUD operations for tasks
- Task assignment
- Task status updates
- Tagging functionality
- Filtering and sorting tasks

## Prerequisites

- Node.js (v22+)
- PostgreSQL
- npm or yarn
